08 · Bài học
Lessons Learned

Những bài học từ dự án

1

Vibe Coding = Speed to prototype, không thay thế planning

AI giúp code nhanh hơn 5–10x, nhưng vẫn cần architect rõ ràng trước. Không có plan → context window của AI bị phân tán, code inconsistent.

2

YAGNI thực sự tiết kiệm thời gian

Phase 1 chỉ làm đúng core feature. Không build backend vì chưa cần. Client-only approach giảm complexity 80%, deploy trong 10 phút.

3

Custom hooks = Game changer cho tổ chức code React

Tách mỗi concern ra một hook (useMealHistory, useGeolocation...) → App.tsx sạch, dễ test, dễ debug.

4

AI API key trên client-side là rủi ro cần nhận thức

Gemini key visible trong network tab. Mitigation: restrict by HTTP referrer trên Google Cloud Console. Phase 4 nên move sang backend proxy.

5

Red team bắt được những gì dev chủ quan bỏ qua

AI code reviewer phát hiện: thiếu try-catch trên localStorage.parse, OAuth token không revoke sau khi dùng, missing fallback khi geolocation timeout.
